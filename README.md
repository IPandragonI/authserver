# AuthServer
Projet de Master 2 — objectif :

SSO & Spring  : créez une authentification centralisée de A à Z

## Description
Projet pédagogique (Master 2) qui illustre la conception d'un Single Sign‑On (SSO) en Java Spring Boot, avec migration Liquibase et configuration Maven. 
Le code principal se trouve dans le répertoire `app`.
La partie cours se trouve dans le répertoire `courses`.

## Prérequis
- Java 17+
- Maven
- Docker
- Python + pip (pour MkDocs si vous voulez servir la doc localement)

# Lancer le serveur mkdocs

> mkdocs serve
