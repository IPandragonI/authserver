Avant de commencer ce cours, nous vous proposons un petit test de positionnement.
Il comprend des questions de différents niveaux de difficulté à travers 50 questions sur Java, Java Spring et Docker.
Vous avez 1h pour le compléter.
Chaque question rapporte 1 point.
A la fin du test, vous obtiendrez un score sur 50. Si vous obtenez un score supérieur ou égal à 40, vous pouvez passer directement au [module 2](../../s2/docker/index.md) même si nous vous le déconseillons.

> Ce test n'est pas noté et n'a pas d'impact sur votre progression dans le cours.
> Il est là pour vous aider à évaluer votre niveau et à identifier les domaines dans lesquels vous pourriez avoir besoin de vous améliorer.

> ⚠️ Ce test est à faire une seule fois. Si vous le refaites, votre score ne sera pas pris en compte.
> Bonne chance !
